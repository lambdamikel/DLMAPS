In order to load the RACE system, start your Common Lisp and
load the file load-race.lisp

**Attention**

It is important that you load the file load-race.lisp from the Lisp Listener.
Do not evaluate or compile the file.

Currently, RACE is supported for

Macintosh Common Lisp 4.2
Allegro Common Lisp 4.3 Solaris
Allegro Common Lisp 5.0 Solaris
Allegro Common Lisp 5.0.1 Windows
Allegro Common Lisp 5.0 Linux
