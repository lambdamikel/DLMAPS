The Description Logic Benchmark Suite

With contributions from Volker Haarslev, Ian Horrocks, Ralf Möller and Peter Patel-Schneider.

In order to run the benchmarks, please decide which DL system to use and follow the instructions below.

  • DLP: Instructions supplied RSN.
  • FaCT:
    1. Install and start your favorite Common Lisp system.
    2. Load the file "fact-tests.lisp" into your Lisp system.
    3. If not compiled beforehand, compile the file "dl-test:FaCT-1.11;fact" and load the corresponding binary file manually. Compiling Fact is strictly recommended, once this third step has been completed, it can be omitted.
    4. Run the tests by evaluating (run-dl-tests). See the definition of this function in the file "fact-tests.lisp" in order to select specific tests.
    5. In order to run the TANCS'99 benchmarks, decompress and extract the problems in the archives found in "dl-test:TANCS99-tests;". Load "dl-test:TANCS99-tests;tancs-tests.lisp" and execute (run-tancs-tests). See the definition of this function in order to select specific tests.
  • RACE:
    1. Install and start your Common Lisp system.
    2. Load the file "race-tests.lisp" into your Lisp system.
    3. Run the tests by evaluating (run-dl-tests). See the definition of this function in the file "race-tests.lisp" in order to select specific tests.
    4. In order to run the TANCS'99 benchmarks, decompress and extract the problems in the archives found in "dl-test:TANCS99-tests;". Load "dl-test:TANCS99-tests;tancs-tests.lisp" and execute (run-tancs-tests). See the definition of this function in order to select specific tests.

You can find the results of various tests in the folder "dl-test:Results;". For further documentation, see the folder "dl-test:Documentation;". Optimization switches can be found in the user guides of the DL systems. If your DL system implementation is not mentioned here and you would like it to be included into the DL Benchmark Suite, please send us a message.
